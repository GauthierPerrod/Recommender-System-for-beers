The goal of this challenge is develop a Recommender System to optimize the prediction of the best beer to suggest to any user.

#### Set up

Open a terminal and

1. install the `ramp-workflow` library (if not already done)
  ```
  $ pip install git+https://github.com/paris-saclay-cds/ramp-workflow.git
  ```
  
2. Follow the ramp-kits instructions from the [wiki](https://github.com/paris-saclay-cds/ramp-workflow/wiki/Getting-started-with-a-ramp-kit)

#### Local notebook

Get started on this RAMP with the [dedicated notebook](beer_rec_ramp_startingkit.ipynb).

To test the starting-kit, run


```
ramp_test_submission --submission starting_kit
```
or for a test mode :

```
ramp_test_submission --submission starting_kit --quick-test
```
